# K-Gigs Platform

K-Gigs eshte nje platforme per lidhjen e freelancer-eve me punedhenesit.  
Punedhenesit mund te postojne pune (gigs), ndersa freelancer-et mund te aplikojne per to.

---

## Teknologjite e perdorura

### Frontend
- React.js
- Tailwind CSS

### Backend
- Node.js
- Express.js

### Databaza
- MySQL

### Autentifikimi
- JSON Web Token (JWT)

---

## Funksionalitetet kryesore

### Users
- Regjistrim dhe login
- Role: freelancer, punedhenes, admin
- Admin mund te:
  - krijoje user
  - perditesoje user
  - fshije user
  - aktivizoje/deaktivizoje user

---

### Gigs (Pune)
- Punedhenesi mund te:
  - krijoje gig
  - perditesoje gig
  - fshije gig
  - shohe aplikimet per gig

---

### CV
- Freelancer mund te:
  - krijoje CV
  - perditesoje CV
  - fshije CV

---

### Applications
- Freelancer mund te:
  - aplikoje per nje gig
  - perditesoje aplikimin
  - terheqe aplikimin

- Punedhenesi mund te:
  - shohe aplikimet
  - pranoje/refuzoje aplikimet

---

### Dashboard
- Freelancer Dashboard
- Punedhenes Dashboard
- Admin Dashboard
